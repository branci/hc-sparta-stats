Záverečná správa

Mojou úlohou v tomto projekte bolo vytvoriť samotné webové rozhranie.

Maven web application

Vytvoril som Maven web application (HcSparta_web), ktorá má nastavenú dependecy na projekte ParsingHTML v ktorom sa údaje zo stránok hcsparta.cz sťahujú, parsujú a ukladajú do databázy.

Servlets

Ďalej som postupoval vytvorením servletov, konkrétne som využíval knižnicu javax.servlet.*, a k nim prislúchajúcich .jsp stránok. V servletoch som implementoval metódy na spracovanie HTTP requestov a ich pomocné metódy.

JSP

Na samotné vytvorenie HTML stránok a následné zobrazenie údajov zo serlvetov som využíval technológiu Java Server Pages. V .jsp listoch som využíval možnosti jazyka JSP EL na rozumné vypisovanie údajov, ktoré vracali metódy z projektu ParsingHTML.
